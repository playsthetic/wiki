# wiki

## Requirements

- [Node.js](https://nodejs.org/) >= 22.0.0

## Development

Install the requirements:

```shell
npm install
```

Run the live server:

```shell
npm run dev
```

## Building

Run the following command:

```shell
npm run build
```

## Deployment

The commits pushed to the `main` branch will automatically be deployed to [GitHub Pages](https://wiki.playsthetic.com/).
